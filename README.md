# Decentralized Blockchain Ledger for Secure Legal Documents

This project is a blockchain-based document management system built using Django and Ethereum.

## 🔹 Features
- Secure document upload and storage
- Blockchain-based metadata storage
- SHA-256 hash verification
- Tamper-proof audit trail
- Search and retrieval system

## 🔹 Tech Stack
- Python
- Django
- Web3.py
- Ethereum (Ganache)
- HTML, CSS

## 🔹 How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run server:
   python manage.py runserver

3. Open browser:
   http://127.0.0.1:8000
